# Burrito Shop API

## Features
- Create new burrito, get a list of burritos
- Create new order given a burrito, get all orders, get order by ID
- Issue loyalty points on Ethereum blockchain when ordering

## Prerequisites
- Node.js
- Docker
- pnpm

## Setup
`pnpm install`

## Running
`pnpm run dev`

## Building
`pnpm run build`

## Test
`pnpm run test`
